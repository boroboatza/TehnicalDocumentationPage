Getting Started with C++

C++ is a powerful and widely-used programming language that offers high-level abstraction and low-level control. It was developed as an extension of the C programming language and provides additional features such as object-oriented programming (OOP) and generic programming.

In this chapter, we will introduce the basics of C++ and guide you through the process of setting up a development environment.

What is C++?

C++ is a statically typed, compiled programming language designed for system programming, embedded systems, and general-purpose software development.

It combines the features of both high-level and low-level languages, making it suitable for a wide range of applications.

Setting up the Development Environment

To write and run C++ programs, you need a C++ compiler and an integrated development environment (IDE). There are several options available, such as Visual Studio, Code::Blocks, and Eclipse. In this example, we will use the GNU Compiler Collection (GCC) along with a text editor like Visual Studio Code.

Installing GCC

To install GCC on your system, follow the appropriate instructions for your operating system. For Linux, you can use the package manager. For Windows, you can download and install MinGW, which includes GCC.

Setting up Visual Studio Code

  1. Install Visual Studio Code from the official website.
  2. Open Visual Studio Code and install the C++ extension by Microsoft.
  3. Create a new folder for your C++ projects and open it in Visual Studio Code.
  4. Create a new file with the .cpp extension and start writing your C++ code.
  5. Start Coding!

Hello, World!

Let's begin with the traditional "Hello, World!" program, which simply displays the text "Hello, World!" on the screen. This program serves as a simple introduction to the basic syntax of C++.

#include <iostream>

int main() 
{
  std::cout << "Hello, World!" << std::endl;
  return 0;
}

In this program, we include the iostream header file, which provides input and output streams.

The main() function is the entry point of every C++ program.

The statement std::cout << "Hello, World!" << std::endl;prints the text "Hello, World!" to the standard output.

Finally, return 0; indicates successful program termination.

Variables and Data Types

In this chapter, we will explore variables and data types in C++.

Variables allow us to store and manipulate data, while data types define the nature and range of the data that can be stored in variables.

Variables

In C++, variables are named storage locations that hold values.

Before using a variable, you must declare its type and name.

Here's an example of declaring and initializing variables:

#include <iostream>

int main() 
{
    // Variable declaration and initialization
    int age = 25;
    double pi = 3.14159;
    char grade = 'A';

    // Output variable values
    std::cout << "Age: " << age << std::endl;
    std::cout << "Pi: " << pi << std::endl;
    std::cout << "Grade: " << grade << std::endl;

    return 0;
}

In this code snippet, we declare and initialize three variables: age of type int, pi of type double, and grade of type char.

We then output the values of these variables using std::cout.

Basic Data Types

C++ provides several basic data types, such as int, double, char, bool, and more.

These data types determine the range and precision of values that can be stored in variables. Here's an example:

#include <iostream>

int main() 
{
    int myInt = 42;
    double myDouble = 3.14;
    char myChar = 'A';
    bool myBool = true;

    std::cout << "Integer: " << myInt << std::endl;
    std::cout << "Double: " << myDouble << std::endl;
    std::cout << "Character: " << myChar << std::endl;
    std::cout << "Boolean: " << myBool << std::endl;

    return 0;
}

In this example, we declare variables of different data types and assign values to them. We then output the values using std::cout.

Control Structures

Control structures allow us to modify the flow of execution in a C++ program.

In this chapter, we will explore decision-making and looping structures that enable us to make choices and repeat code blocks.

If-Else Statement

The if-else statement is used to perform conditional execution in C++. It allows us to execute different blocks of code based on a given condition.

Here's an example:

#include <iostream>

int main() 
{
    int age = 18;

    if (age >= 18) 
    {
        std::cout << "You are eligible to vote." << std::endl;
    } 
    else 
    {
        std::cout << "You are not eligible to vote." < std::endl;
    }
    return 0;
}

In this code, we check if the variable age is greater than or equal to 18 using the if statement.

If the condition is true, the first block of code is executed; otherwise, the code inside the else block is executed.

Switch Statement

The switch statement provides an alternative way to perform multiway branching based on the value of an expression. It allows us to execute different blocks of code based on different possible values.

Here's an example:

#include <iostream>

int main() 
{
    int day = 3;

    switch (day) 
    {
        case 1:
            std::cout << "Monday" << std::endl;
            break;
        case 2:
            std::cout << "Tuesday" << std::endl;
            break;
        case 3:
            std::cout << "Wednesday" << std::endl;
            break;
        default:
            std::cout << "Invalid day" << std::endl;
            break;
    }

    return 0;
}

In this code, the value of the variable day is checked against different cases using the switch statement.

If the value matches a case, the corresponding block of code is executed.

The break statement is used to exit the switch block once a case is matched.

Functions

Functions in C++ are reusable blocks of code that perform specific tasks. They allow us to break our programs into smaller, modular pieces.

In this chapter, we will explore function declaration, definition, and calling.

Function Declaration

To use a function in C++, you must first declare it. Function declarations provide the necessary information about the function's name, return type, and parameters.

Here's an example:

#include <iostream>

// Function declaration
int addNumbers(int a, int b);

int main() 
{
    int num1 = 5;
    int num2 = 7;

    // Function calling
    int sum = addNumbers(num1, num2);
    std::cout << "Sum: " << sum << std::endl;

    return 0;
}

// Function definition
int addNumbers(int a, int b) 
{
    return a + b;
}

In this code, we declare the addNumbers() function before the main() function.

The function declaration specifies that addNumbers() takes two integers as parameters and returns an integer.

The function definition is placed after the main() function and contains the actual implementation of the function.

Function Calling

To call a function in C++, you simply use its name followed by parentheses and any required arguments.

Here's an example:

#include <iostream>

// Function declaration
void greet();

int main() 
{
    // Function calling
    greet();

    return 0;
}

// Function definition
void greet() 
{
    std::cout << "Hello, world!" << std::endl;
}

In this code, we declare and define the greet() function, which doesn't take any arguments and doesn't return a value.

Inside the main() function, we call greet() to execute the code inside it.

Arrays

Arrays in C++ are used to store multiple values of the same data type in contiguous memory locations.

In this chapter, we will explore arrays, their declaration, initialization, and accessing elements.

Array Declaration and Initialization

To declare an array in C++, you specify the data type of the elements followed by the name of the array and the size within square brackets.

Here's an example:

#include <iostream>

  int main() 
  {
      // Array declaration and initialization
      int numbers[5] = {1, 2, 3, 4, 5};

      return 0;
  }

Note: Array indexes start with 0: [0] is the first element. [1] is the second element, etc.

In this code, we declare an integer array named numbers with a size of 5.

We initialize the array with the values 1, 2, 3, 4, and 5.

Accessing Array Elements

You can access individual elements of an array using the array name followed by the index within square brackets.

Array indices start from 0 for the first element and go up to size-1 for the last element.

Here's an example:

#include <iostream>

int main() 
{
    int numbers[5] = {1, 2, 3, 4, 5};

    // Accessing array elements
    std::cout << numbers[0] << std::endl;  // Output: 1
    std::cout << numbers[2] << std::endl;  // Output: 3

    return 0;
}

In this code, we access the first and third elements of the numbers array using the indices 0 and 2, respectively.

Pointers

Pointers are variables that store memory addresses.

They enable us to manipulate memory directly and facilitate dynamic memory allocation.

In this chapter, we will explore pointers, their declaration, initialization, and usage.

Pointer Declaration and Initialization

o declare a pointer in C++, you use the * symbol before the pointer name.

Pointers must be initialized with the memory address of a variable.

Here's an example:

#include <iostream>

int main() 
{
    int number = 42;
    int* ptr = &number;

    return 0;
}

In this code, we declare an integer variable number and initialize it with the value 42

We also declare a pointer ptr and initialize it with the memory address of number using the & operator.

Dereferencing Pointers

To access the value stored at a memory address pointed by a pointer, we use the dereference operator * .

Here's an example:

#include <iostream>

int main() 
{
    int number = 42;
    int* ptr = &number;

    // Dereferencing the pointer
    std::cout << "Value: " << *ptr << std::endl;  // Output: 42

    return 0;
}

In this code, we dereference the ptr pointer using *ptr to obtain the value stored at the memory address it points to.

OOP

C++ supports object-oriented programming (OOP) concepts such as classes, objects, inheritance, and polymorphism.

In this chapter, we will introduce the basics of OOP in C++.

Classes and Objects

In C++, a class is a blueprint for creating objects. It defines the properties (data members) and behaviors (member functions) of objects

Here's an example of a class:

#include <iostream>

class Rectangle 
{
    int width;
    int height;

public:
    // Constructor
    Rectangle(int w, int h) 
    {
        width = w;
        height = h;
    }

    // Member function
    int getArea() 
    {
        return width * height;
    }
};

int main() 
{
    // Creating objects of the class
    Rectangle rect1(4, 5);
    Rectangle rect2(3, 6);

    // Accessing member functions
    std::cout << "Area of rect1: " << rect1.getArea() << std::endl;
    std::cout << "Area of rect2: " << rect2.getArea() << std::endl;

    return 0;
}

In this code, we define a Rectangle class with private data members width and height and a public member function getArea()

We create objects rect1 and rect2 of the class and access their member functions.

Inheritance

Inheritance is a fundamental concept in OOP that allows a class to inherit the properties and behaviors of another class.

It promotes code reuse and enables the creation of hierarchies of classes.

Here's an example:

#include <iostream>

// Base class
class Shape {
protected:
    int width;
    int height;

public:
    // Constructor
    Shape(int w, int h) {
        width = w;
        height = h;
    }

    // Member function
    void displayArea() {
        std::cout << "Area: " << width * height << std::endl;
    }
};

// Derived class
class Rectangle : public Shape {
public:
    // Constructor
    Rectangle(int w, int h) : Shape(w, h) {}

    // Additional member function
    void displayPerimeter() {
        std::cout << "Perimeter: " << 2 * (width + height) << std::endl;
    }
};

int main() {
    // Creating an object of the derived class
    Rectangle rect(4, 5);

    // Accessing member functions from the base and derived classes
    rect.displayArea();
    rect.displayPerimeter();

    return 0;
}

In this code, we define a base class Shape with private data members width and height and a member function displayArea()

We then create a derived class Rectangle that inherits from Shape using the public keyword.

The derived class adds an additional member function displayPerimeter()

e create an object rect of the derived class and access member functions from both the base and derived classes.

Files

File input/output (I/O) operations allow us to read data from and write data to files.

In this chapter, we will explore file I/O in C++ using file streams.

Writing to a File

To write data to a file in C++, we use the output file stream ofstream class.

Here's an example:

#include <iostream>
#include <fstream>

int main() {
    std::ofstream outputFile("output.txt");

    if (outputFile.is_open()) {
        outputFile << "Hello, world!" << std::endl;
        outputFile << 42 << std::endl;
        outputFile.close();
        std::cout << "Data written to the file." << std::endl;
    } else {
        std::cout << "Failed to open the file." << std::endl;
    }

    return 0;
}

In this code, we create an ofstream object named outputFile and open the file "output.txt"

We then use the << operator to write data to the file.

Finally, we close the file and display a message indicating the success or failure of the operation.

Reading from a File

To read data from a file in C++, we use the input file stream ifstream class.

Here's an example:

#include <iostream>
#include <fstream>
#include <string>

int main() 
{
    std::ifstream inputFile("input.txt");
    std::string line;

    if (inputFile.is_open()) 
    {
        while (std::getline(inputFile, line)) 
        {
            std::cout << line << std::endl;
        }
        inputFile.close();
    } 
    else 
    {
        std::cout << "Failed to open the file." << std::endl;
    }

    return 0;
}

In this code, we create an ifstream object named inputFile and open the file "input.txt".

We use the std::getline() function to read each line from the file and display it on the console.

Finally, we close the file.

Exception Handling

Exception handling allows us to handle runtime errors and abnormal conditions in our code.

It provides a mechanism to catch and handle exceptions, preventing program termination.

In this chapter, we will explore exception handling in C++ using try-catch blocks.

Exception Handling with Try-Catch

To handle exceptions in C++, we use the try-catch block.

The try block contains the code that may throw an exception, while the catch block handles the thrown exception.

Here's an example:

#include <iostream>

int main() {
    try {
        int num1 = 10;
        int num2 = 0;

        if (num2 == 0) {
            throw "Divide by zero exception.";
        }

        int result = num1 / num2;
        std::cout << "Result: " << result << std::endl;
    } catch (const char* message) {
        std::cout << "Exception caught: " << message << std::endl;
    }

    return 0;
}

In this code, we divide num1 by num2 and check if num2 is zero.

If it is, we throw an exception with the message "Divide by zero exception.".

The catchcatch block catches the exception and displays the error message.

Exception Handling with Multiple Catch Blocks

In C++, we can have multiple catch blocks to handle different types of exceptions. This allows us to handle specific exceptions separately.

Here's an example:

#include <iostream>

int main() 
{
    try 
    {
        int num1 = 10;
        int num2 = 0;

        if (num2 == 0) 
        {
            throw "Divide by zero exception.";
        }

        int result = num1 / num2;
        std::cout << "Result: " << result << std::endl;
    } 
    
    catch (const char* message) 
    {
        std::cout << "Exception caught: " << message << std::endl;
        
    }
    
    catch (...) 
    {
        std::cout << "Unknown exception caught." << std::endl;
    }

    return 0;
}

In this code, we add an additional catch block with ellipsis (...) to catch any unknown exceptions that were not caught by the previous catch block.

STL

The Standard Template Library (STL) is a powerful library in C++ that provides a collection of reusable algorithms and data structures.

In this chapter, we will explore some components of the STL, such as vectors, iterators, and algorithms.

Vectors

Vectors are dynamic arrays in C++ that can grow or shrink in size.

They provide a convenient way to store and manipulate collections of elements.

Here's an example:

#include <iostream>
#include <vector>

int main() {
    std::vector<int< numbers;

    // Adding elements to the vector
    numbers.push_back(1);
    numbers.push_back(2);
    numbers.push_back(3);

    // Accessing elements
    std::cout << "First element: " << numbers[0] << std::endl;
    std::cout << "Size: " << numbers.size() << std::endl;

    // Iterating over the vector
    for (int i = 0; i < numbers.size(); i++) 
    {
        std::cout << numbers[i] << " ";
    }
    std::cout << std::endl;

    return 0;
}

In this code, we create a vector numbers to store integers.

We use the push_back() function to add elements to the vector.

We access elements using the square bracket operator [] and retrieve the size of the vector using the size() function.

Finally, we iterate over the vector using a for loop and display its elements.

Iterators

Iterators in C++ provide a way to traverse and manipulate the elements of a container, such as a vector.

They act as pointers to elements in the container and provide a uniform interface for accessing and modifying the elements.

Here's an example:

#include <iostream>
#include <vector>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};

    // Using iterators to access and modify elements
    std::vector<int>::iterator it;
    for (it = numbers.begin(); it != numbers.end(); it++) {
        std::cout << *it << " ";
        *it *= 2;
    }
    std::cout << std::endl;

    // Displaying modified elements
    for (const auto& num : numbers) {
        std::cout << num << " ";
    }
    std::cout << std::endl;

    return 0;
}

In this code, we create a vector numbers and initialize it with values.

We use an iterator it to traverse the vector using the begin() and end() functions.

We dereference the iterator using *it to access and modify the elements.

Finally, we use a range-based for loop to display the modified elements.

Algorithms

The STL provides a wide range of algorithms that can be applied to containers, such as vectors.

These algorithms include sorting, searching, transforming, and more.

Here's an example:

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> numbers = {5, 2, 8, 1, 4};

    // Sorting the vector
    std::sort(numbers.begin(), numbers.end());

    // Displaying the sorted vector
    for (const auto& num : numbers) {
        std::cout << num << " ";
    }
    std::cout << std::endl;

    // Finding an element in the vector
    auto it = std::find(numbers.begin(), numbers.end(), 8);
    if (it != numbers.end()) {
        std::cout << "Element found at position: " 
                  << std::distance(numbers.begin(), it) 
                  << std::endl;
    } 
    else 
    {
        std::cout << "Element not found." << std::endl;
    }

    return 0;
}

In this code, we create a vector numbers and initialize it with values.

We use the std::sort() algorithm to sort the vector in ascending order.

We display the sorted vector using a range-based for loop.

We then use the std::find() algorithm to search for the element 8 in the vector and display its position if found.
